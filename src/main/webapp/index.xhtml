A cifra Vigenère é um método de criptografia que usa uma série de diferentes "Cifras de César" com base nas letras de uma palavra-chave. Em uma cifra César, cada letra na passagem é movida para um certo número de letras, para ser substituído pela letra correspondente. Por exemplo, isso significaria que em uma mudança de César de três: A se tornaria D; B se tornaria E; C se tornaria F, etc. Uma cifra de Vigenère baseia-se neste método usando múltiplas cifras de César em diferentes pontos da mensagem.
